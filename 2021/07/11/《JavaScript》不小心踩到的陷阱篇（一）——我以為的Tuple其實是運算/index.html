《JavaScript》不小心踩到的陷阱篇（一）——我以為的Tuple其實是運算

發表於 2021-07-11 分類於 JavaScript

請問上述程式碼會印出什麼呢？

答案是2。不知道是否跟你心目中想像的結果一樣嗎？

逗號運算子

從MDN可以查到JavaScript對於逗號的定義：

The comma operator evaluates each of its operands (from left to right) and returns the value of the last operand.
MDN

在JavaScript中的逗號其實是運算子，他會由左至右計算每一個運算元的值，並回傳最後一個運算元計算後的值。所以，並不像其他語言（例如：C#），小括號裡面放個逗號表示Tuple，可別搞混啦～

小試身手

來個小題目確定自己搞懂了吧～

上述程式碼會印出什麼呢？

答案是7。
解說：一開始a=1，第二行由左至右運算，a++可以視a+=1，所以此時a=2。接下來計算a-10，但此結果並沒有回傳值，只是個純運算。最後計算a+5，得到結果為7。三個運算元計算完畢，回傳最後一個運算元的值給b。最終印出b就是7啦。
